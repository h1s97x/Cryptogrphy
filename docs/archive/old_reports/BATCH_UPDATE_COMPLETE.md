## 已知问题

### 1. KeyGroup组件访问方式

**问题**：批量更新脚本无法自动识别哪些组件应该使用KeyGroup

**影响**：部分组件可能仍在Group中使用Key配置

**解决方案**：
- 手动检查每个组件
- 将Key配置从Group移到KeyGroup
- 更新访问方式：`.get_text()` → `.text()`

### 2. QLineEdit的set_text方法

**问题**：QLineEdit使用`.setText()`而不是`.set_text()`

**影响**：部分组件可能调用错误的方法

**解决方案**：
- 手动检查KeyGroup中的组件
- 将`.set_text()` 改为 `.setText()`

### 3. 测试覆盖不足

**问题**：只测试了3个组件

**影响**：其他组件可能存在未发现的问题

**解决方案**：
- 创建自动化测试脚本
- 逐个测试所有组件
- 修复发现的问题
