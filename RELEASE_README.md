# 密码学平台 v2.2.0

一个集成了多种密码算法和协议的教学演示平台，采用现代化的 Fluent Design 界面。

## 📦 快速开始

1. 解压 `密码学平台.zip`
2. 双击 `密码学平台.exe`
3. 开始使用！

## ✨ 功能特性

### 35个算法和协议

- **经典密码** (7个): Caesar, Hill, Vigenere, Playfair, Enigma...
- **对称密码** (7个): AES, DES, SM4, RC4, SPECK, SIMON...
- **公钥密码** (4个): RSA, RSA Sign, ElGamal, ECDSA
- **哈希算法** (7个): MD5, SHA-1, SHA-256, SHA-3, SM3...
- **数学基础** (3个): Euler, CRT, Euclidean
- **密码协议** (7个): 重放攻击, 零知识证明, 数字证书...

### 现代化界面

- Fluent Design 设计语言
- 浅色/深色主题切换
- 清晰的卡片式布局
- 实时操作日志

### 教学功能

- 算法原理介绍（HTML页面）
- 完整的加密/解密流程演示
- 参数可调整
- 结果可视化

## 💻 系统要求

- Windows 10/11 (64位)
- 无需安装Python
- 无需安装依赖库

## 📖 使用说明

详细使用说明请查看 `USER_GUIDE.md`

### 基本操作

1. **选择算法**: 在首页点击分类卡片，然后选择具体算法
2. **输入数据**: 在输入框中输入明文或密文
3. **设置参数**: 根据算法要求设置密钥等参数
4. **执行操作**: 点击"加密"或"解密"按钮
5. **查看结果**: 在输出框和日志区域查看结果

### 主题切换

点击左下角"设置" → 选择主题（浅色/深色/自动）

### 算法介绍

部分算法提供详细介绍，点击右上角"📖 算法介绍"按钮查看

## ⚠️ 注意事项

### 首次启动

- 首次启动需要10-20秒，请耐心等待
- Windows Defender 可能会扫描（正常现象）

### 杀毒软件

如果被杀毒软件拦截：
- 这是误报，程序是安全的
- 请添加到白名单

### 性能

- 某些操作（如大数运算）可能需要几秒钟
- 请查看操作日志了解进度
